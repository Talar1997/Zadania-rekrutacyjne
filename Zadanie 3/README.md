Napisz w dowolnym języku program, który wyświetli na konsoli następujący ciag znaków (choinkę):
```
            *
           ***
          *****
         *******
        *********
       ***********
```
Liczba pięter powinna być parametryzowana. Preferowane rozwiązanie bez użycia pętli. 
```
Iteracja -> Ilość * w linii
1 -> 1 + 0
2 -> 2 + 1
3 -> 3 + 2
4 -> 4 + 3
5 -> 5 + 4
n -> n + (n-1)
```
